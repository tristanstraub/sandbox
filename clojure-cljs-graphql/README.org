GraphQL client/server example

* Server

clj -m graphql.server

Visit: http://localhost:8080/graphiql to use GraphiQL

Example queries (take from https://github.com/walmartlabs/lacinia)

#+BEGIN_SRC graphql
{
  hero {
    id
    name
  }
}
#+END_SRC

#+BEGIN_SRC graphql
{
  hero(episode: NEWHOPE) {
    movies: appears_in
  }
}
#+END_SRC
